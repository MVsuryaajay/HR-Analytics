# HR Analytics PowerBI Dashboard

**Developed by: Musinada Venkata Surya Ajay**

## Project Overview
This project focuses on HR Data Analytics, specifically targeting key performance indicators (KPIs) such as employee attrition rate, performance tracking, and company demographics. The goal was to provide actionable insights for HR departments to improve employee retention and performance management.

### Project Explanation
- **Data Source**: Analyzed a dataset of over 80,000 HR records (processed down to 76,000 cleaned records).
- **Tools Used**: 
  - **Microsoft Excel & Power BI Query Editor**: Used for extensive data cleaning, transforming raw data into a usable format, and handling missing values.
  - **Microsoft Power BI**: Used to create a dynamic, multi-page dashboard for visualization.
- **Key Features**:
  - **Overview Dashboard**: High-level KPIs including total employees, attrition count, and attrition rate.
  - **Demographics**: Visualization of the workforce by age, gender, and department.
  - **Performance Tracker**: Analysis of employee performance ratings across teams.
  - **Attrition Analysis**: Deep dive into reasons for attrition and trends over time.

## Key Insights
- Identified departments with high attrition rates, allowing for targeted retention strategies.
- Visualized performance trends against demographics to ensure unbiased performance appraisals.
